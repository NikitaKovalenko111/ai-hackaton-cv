# AI Hackaton

Проект был создан в рамках ИИ Хакатона университета УУНиТ. Обучена модель Yolo на сегментацию растений рукколы/пшеницы. Нейросеть сегментирует растение на лепестки, стебель, корень и автоматически подсчитывает длину и площадь всех сегментов, а также классифицирует растение. Датасет был предоставлен УФИЦ РАН.

## Характеристики модели

- **Box**:
    - **Precision**: 0.963

    - **Recall**: 0.971

    - **Mask50**: 0.964

    - **Mask50-95**: 0.811

- **Mask**:
    - **Precision**: 0.963

    - **Recall**: 0.971

    - **Mask50**: 0.962

    - **Mask50-95**: 0.664

## Установка

### 1. Проверка работы

Проверить работу сайта можно по [ссылке](http://94.41.18.199:3000/) или в телеграм боте @Al_plantbot (телеграм может сжимать изображение, из-за чего длины могут считаться некорректно, предпочтительнее использовать сайт), (сервер может иногда быть выключен, сообщите, чтобы включили)

Телеграм бот может плохо работать из-за замедления телеграм

### 2. Установка через Docker

Для установки через docker вам понадобится docker и docker-compose:

2.1 Переместите датасет в папку model

2.2 Выполните аугментацию через скрипт model/utils/augmentation.py

2.3 Обучите модель через скрипт model/train.py

2.4 Переместить модель в папку model/runs/segment

2.5 Укажите пути до модели в файле model/server.py

2.6 Для запуска бота требуется создать в папке /bot файл config.py и записать туда BOT_TOKEN=<токен бота>

2.7 Запустить docker compose

```
docker compose up --build
```

в корне проекта

2.8 После запуска дождаться, пока подготовится датасет для классификатора

#### Для запуска frontend нужно:

2.11 В файле client/src/api/api.ts поменять ссылку на свой сервер

2.12 Собрать проект

```
npm run build
```

2.13 Переместить содержимое папки dist в var/www/_название_сайта_

**У вас должен быть настроен nginx**

ИЛИ

2.12 Запустить frontend

```
npm run start
```

### 3. Пошаговая установка

3.1 Установить venv и зайти в нее

```
python -m venv .venv

.venv\Scripts\Activate
```

в папке **model**

3.2 Установить pytorch

```
pip install torch==2.7.1 torchvision==0.22.1 torchaudio==2.7.1 --index-url https://download.pytorch.org/whl/cu118
```

3.3 Установить ultralytics и thop

```
pip install ultralytics==8.4.18 ultralytics-thop==2.0.18
```

3.4 Установить базовые библиотеки

```
pip install -r requirements.txt
```

3.5 Загрузите датасет в папку model (папка dataset)

3.6 Выполните аугментацию (augmentation.py)

3.7 Для обучения модели запустите train.py

3.8 После обучения переместите папку модели в model/runs/segment

3.9 В server.py поменяйте путь до модели

3.10 Запустить server.py

3.11 Дождаться подготовки датасета для классификатора

3.12 Запустить бота (bot/main.py), если требуется.

Для запуска бота требуется создать в папке /bot файл config.py и записать туда BOT_TOKEN=<токен бота>

3.13 Запустить frontend

```
npm install

npm run start
```

в папке client
